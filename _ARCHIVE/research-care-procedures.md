# 介護関連行政手続き・制度 調査まとめ

> 調査日: 2026-03-07
> 対象: ワーキングケアラー・介護当事者が知るべき主要制度

---

## 1. 介護保険（要介護・要支援認定）

### 概要
65歳以上（または40〜64歳で特定疾病がある人）が介護サービスを利用するための基本制度。要介護度に応じてサービス利用限度額が決まる。

### 対象者
- 第1号被保険者：65歳以上
- 第2号被保険者：40〜64歳で国が定める16種類の特定疾病（がん末期・関節リウマチ・パーキンソン病等）が原因の場合に限る

### A. 新規申請の手続きステップ

1. **窓口に申請**
   - 住所地の市区町村の介護保険課・福祉課 または 地域包括支援センター
   - 家族や指定居宅介護支援事業者が代理申請も可
2. **認定調査（訪問調査）**
   - 市区町村の調査員が自宅等を訪問し、心身状態を約1時間ヒアリング
3. **主治医意見書の作成**
   - 市区町村が主治医へ依頼（申請者が医師情報を記入）
4. **一次判定**
   - コンピューターが調査結果をもとに要介護度を仮判定
5. **二次判定（介護認定審査会）**
   - 保健・医療・福祉の専門家が審査会で最終判定
6. **認定結果通知**
   - 申請日から原則30日以内に通知書と被保険者証が郵送
7. **ケアプラン作成・サービス利用開始**
   - 要支援→地域包括支援センター、要介護→居宅介護支援事業者（ケアマネ）

### B. 更新申請

- 認定有効期間が終わる前に更新申請が必要（有効期間は初回6ヶ月〜最長48ヶ月）
- 手続きは新規申請とほぼ同じ
- 有効期間満了日の60日前から受付可能（市区町村から通知が届く）

### C. 区分変更申請

- 認定有効期間中に心身状態が著しく変化した場合に申請
- 手続きは新規申請と同じ（調査→審査→判定）
- 注意：変更申請の結果、現在の区分より下がる可能性もある

### 必要書類
| 書類 | 備考 |
|------|------|
| 要介護・要支援認定申請書 | 窓口またはHP から入手 |
| 被保険者証（介護保険証） | 第1号は保険者が発行済み |
| 本人確認書類 | マイナンバーカード、運転免許証等（写真付き1点 or 写真なし2点） |
| マイナンバー確認書類 | 写しでも可 |
| （第2号）医療保険被保険者証 | 健康保険証 |

### 申請先
住所地の市区町村の介護保険課・高齢者支援課・福祉課（窓口名称は自治体による）
または地域包括支援センター（代行可）

### 注意点・よくある間違い
- **申請日が基準日**になるため、「まだ早い」と思わず早めに申請する
- 主治医が「かかりつけ医」でない場合は別途医師を探す必要がある
- 調査当日は本人の普段の状態を正確に伝えることが重要（調子の良い日の状態で答えると低く判定されがち）
- 認定結果に不満がある場合は、都道府県の介護保険審査会に不服申立てが可能

## 2. 障害者手帳

### A. 身体障害者手帳

#### 概要
身体に永続する障害のある人が受けられる手帳。1〜6級があり、等級に応じて各種サービス・税制優遇を受けられる。

#### 対象者
視覚・聴覚・言語機能・肢体不自由・心臓・腎臓・呼吸器等の内部障害で、永続する機能障害がある人

#### 手続きステップ
1. 市区町村の障害福祉担当窓口で申請書類を受け取る
2. 都道府県指定の「身体障害者福祉法第15条指定医」に診断書・意見書の作成を依頼
3. 申請書・診断書・写真・マイナンバー関係書類をそろえて窓口に提出
4. 都道府県（心身障害者福祉センター等）が審査
5. 手帳交付（通常1ヶ月程度）

#### 必要書類
| 書類 | 備考 |
|------|------|
| 身体障害者手帳交付申請書 | 窓口で入手 |
| 身体障害者診断書・意見書 | 発行から1年以内、指定医作成 |
| 写真（縦4cm×横3cm） | 上半身・脱帽 |
| マイナンバー確認書類 | 申請書への記載が必要 |
| 本人確認書類 | — |

#### 申請先
お住まいの区市町村の障害福祉担当窓口（福祉事務所・障害福祉課等）

#### 注意点
- 診断書は「指定医」でないと受け付けられない（窓口や市区町村HPで確認）
- 診断書の内容次第で審査に時間がかかることがある
- 等級に不服がある場合は都道府県に審査請求が可能

---

### B. 精神障害者保健福祉手帳

#### 概要
精神疾患（統合失調症・うつ病・発達障害等）で日常生活・社会生活に制約がある人を対象。1〜3級があり、税制優遇・公共交通割引・就労支援等を受けられる。

#### 対象者
精神疾患の初診から6ヶ月以上経過し、日常生活・社会生活に制約がある人

#### 手続きステップ
1. 市区町村の障害福祉担当窓口で申請書類を受け取る
2. 精神科・心療内科の主治医に診断書の作成を依頼（または障害年金受給の場合は年金証書等で代替可）
3. 申請書・診断書・写真・マイナンバー関係書類を窓口に提出
4. 都道府県の精神保健福祉センターが審査（約2ヶ月）
5. 手帳交付

#### 必要書類
| 書類 | 備考 |
|------|------|
| 精神障害者保健福祉手帳交付申請書 | 窓口で入手 |
| 精神障害者保健福祉手帳用診断書 | 主治医作成、初診から6ヶ月以上経過 |
| 写真（縦4cm×横3cm） | 上半身 |
| マイナンバー確認書類 | — |
| 障害年金証書の写し（年金受給者の場合） | 診断書の代わりに使用可 |

#### 申請先
お住まいの市区町村の障害福祉担当窓口

#### 注意点
- 有効期間は2年（2年ごとに更新が必要）
- 診断書不要の場合（年金証書等で申請）は、振込通知書または通帳（直近の振込確認）が必要
- 審査に約2ヶ月かかるため、早めに申請する

## 3. 成年後見制度

### A. 法定後見（補助・保佐・後見）

#### 概要
すでに判断能力が不十分になった人の財産管理・身上保護を支援するため、家庭裁判所が後見人等を選任する制度。

#### 対象者
| 類型 | 対象 | 援助者 |
|------|------|--------|
| 後見 | 判断能力が全くない | 成年後見人 |
| 保佐 | 判断能力が著しく不十分 | 保佐人 |
| 補助 | 判断能力が不十分 | 補助人 |

#### 手続きステップ
1. **管轄家庭裁判所を確認**（本人の住所地を管轄する家庭裁判所）
2. **申立て書類を家庭裁判所で入手**（またはWebからダウンロード）
3. **書類を準備する**
   - 戸籍謄本・住民票の取得
   - 医師の診断書の作成依頼
   - 財産目録の作成
4. **家庭裁判所に申立て**（申立手数料：収入印紙800円＋登記手数料2,600円）
5. **家庭裁判所による調査**（書面審査・本人への面接等）
6. **必要な場合は鑑定**（鑑定料：5万〜10万円程度）
7. **後見開始の審判**（後見人等の選任）
8. **後見登記**（法務局が登記）

#### 必要書類
| 書類 | 備考 |
|------|------|
| 後見開始申立書 | 家庭裁判所書式 |
| 申立事情説明書 | — |
| 親族関係図 | — |
| 財産目録・その資料 | 預貯金通帳等のコピー |
| 本人の戸籍謄本 | — |
| 本人の住民票 | — |
| 申立人の住民票 | — |
| 医師の診断書 | 家庭裁判所所定の書式 |
| 後見人候補者事情説明書 | 候補者がいる場合 |
| 親族意見書 | — |

#### 申請先
本人の住所地を管轄する家庭裁判所

#### 費用
- 申立手数料（収入印紙）：800円
- 登記手数料（収入印紙）：2,600円
- 郵便切手：数千円程度（各裁判所に要確認）
- 鑑定費用：5万〜10万円程度（必要な場合のみ）
- 経済的余裕がない場合は法テラス（0570-078374）に相談可

#### 注意点
- 後見人は家庭裁判所が選任するため、希望した人物にならない場合がある（弁護士・司法書士が選任されることも）
- 後見が始まると本人の法律行為が制限される範囲が広い
- 申立てから審判まで通常1〜4ヶ月かかる

---

### B. 任意後見

#### 概要
本人が判断能力があるうちに、将来の後見人となる人（任意後見受任者）と契約しておく制度。

#### 対象者
現在は判断能力がある成人で、将来に備えたい人

#### 手続きステップ
1. **後見を依頼する人（任意後見受任者）を決める**（家族・弁護士・司法書士等）
2. **公証役場に相談し、任意後見契約公正証書を作成する**
   - 公証人が内容を確認し、公正証書として作成
   - 公証人が法務局（後見登記等ファイル）に登記
3. **本人の判断能力が低下した時点で家庭裁判所に申立て**
   - 任意後見監督人の選任を申立て（本人・配偶者・4親等内親族・任意後見受任者）
4. **家庭裁判所が任意後見監督人を選任**
5. **任意後見契約が効力を発揮し、任意後見人として業務開始**

#### 必要書類（公証役場）
| 書類 | 備考 |
|------|------|
| 本人の実印・印鑑証明書 | — |
| 任意後見受任者の印鑑証明書 | — |
| 本人・受任者の戸籍謄本 | — |
| 住民票 | — |

#### 申請先
- 契約締結：公証役場
- 監督人選任申立て：家庭裁判所

#### 注意点
- 契約の効力は「本人の判断能力が低下し、家庭裁判所が監督人を選任した時点」から発生
- 公正証書作成費用：1万1,000円（財産目録等の作成費用が別途かかる場合あり）
- 任意後見と合わせて「財産管理委任契約」も締結しておくと判断能力低下前にも対応できる

## 4. 介護休業制度・介護休業給付金

### 概要
介護のために仕事を休む労働者の雇用継続を支援する制度。育児・介護休業法に基づき、休業中は雇用保険から給付金が支払われる。

### 対象者
- 雇用保険の被保険者
- 休業開始前2年間に雇用保険の被保険者期間が12ヶ月以上
- 要介護状態（2週間以上常時介護を必要とする）の家族を介護する人

**対象家族の範囲：** 配偶者（事実婚含む）・父母・子・祖父母・兄弟姉妹・孫・配偶者の父母

### 手続きステップ（労働者）

1. **会社への介護休業申出**
   - 原則として休業開始予定日の2週間前までに書面で申出
   - 会社所定の「介護休業申出書」を提出
2. **会社が雇用保険に申請**
   - 事業主が「介護休業給付金支給申請書」をハローワークに提出
   - 申請期限：介護休業終了日の翌日から2ヶ月後の月末まで
3. **給付金受給**
   - 会社の給与口座または本人口座に支給

### 給付金の計算
**支給額 = 休業開始時賃金日額 × 支給日数 × 67%**

- 取得上限：対象家族1人につき通算93日・最大3回まで分割取得可

### 必要書類
| 書類 | 提出先 |
|------|--------|
| 介護休業給付金支給申請書 | ハローワーク（事業主経由） |
| 被保険者が提出した介護休業申出書 | — |
| 住民票記載事項証明書等（対象家族の続柄証明） | — |
| 出勤簿またはタイムカード | — |
| 賃金台帳 | — |

### 申請先
事業主がハローワーク（公共職業安定所）に提出（労働者は会社に申出）

### 2025年改正ポイント（2024年5月成立・2025年4月〜10月施行）
- 会社による介護休業制度の周知徹底・意向確認が義務化
- 雇用6ヶ月未満の労働者も介護休業取得可能に（一部制限の撤廃）
- テレワーク・時差出勤等の柔軟な勤務制度の努力義務化

### 注意点
- **介護休業と介護休暇は別制度**（休暇は1日・半日単位、年5日まで）
- 休業期間中は給与が支払われない場合が多いが、給付金は67%補填
- 申請期限（休業終了翌日から2ヶ月後の月末）を逃すと受給できなくなる

## 5. 高額介護サービス費

### 概要
介護保険サービスを利用した際の1ヶ月の自己負担額が所得に応じた上限額を超えた場合、超過分が払い戻される制度。

### 対象者
要介護認定を受けており、介護保険サービスを利用している人（世帯）

### 負担上限額（月額、2025年時点）

| 区分 | 上限額（月） |
|------|------------|
| 課税所得690万円以上 | 14万100円 |
| 課税所得380〜690万円未満 | 9万3,000円 |
| 課税所得380万円未満（課税世帯） | 4万4,400円 |
| 市区町村民税非課税世帯 | 2万4,600円 |
| 前年収入80万円以下（非課税） | 1万5,000円（個人）／2万4,600円（世帯） |
| 生活保護受給者等 | 1万5,000円 |

### 手続きステップ

1. **初回は市区町村から申請書が自動送付される**（上限額超過が初めて発生した月の翌月に通知）
2. **高額介護サービス費支給申請書に記入し窓口または郵送で提出**
3. **審査後、指定口座に振込**
4. **2回目以降は自動振込（申請不要）**

### 必要書類
| 書類 | 備考 |
|------|------|
| 高額介護サービス費支給申請書 | 市区町村から送付される |
| 介護保険被保険者証 | — |
| 印鑑 | — |
| 振込先口座情報（本人名義） | 通帳またはキャッシュカードの写し |

### 申請先
住所地の市区町村の介護保険課

### 注意点
- **申請期限：サービス提供月の翌月1日から2年間**（期限を超えると受給不可）
- 初回の申請書が届かない場合は自治体窓口に確認を
- 食費・居住費は対象外（高額介護サービス費の対象は利用者負担1〜3割分のみ）
- 高額医療・高額介護合算制度（医療費と介護費の合算）も別途存在する

## 6. 特別障害者手当

### 概要
20歳以上で著しく重度の障害があり、常時特別の介護が必要な人に支給される国の手当。

### 対象者
- **20歳以上**
- 精神または身体に著しく重度の障害があり、日常生活で常時特別の介護を必要とする人
- 在宅の人（施設入所者・3ヶ月超の入院者は対象外）
- 所得制限あり（本人・配偶者・扶養義務者）

### 支給額
**月額 29,590円**（令和7年4月より適用）
支給月：2月・5月・8月・11月（各前3ヶ月分をまとめて支給）

### 手続きステップ

1. **市区町村の障害福祉担当窓口で申請書類を受け取る**
2. **医師に診断書を作成してもらう**（障害の種類ごとに所定様式）
3. **必要書類を窓口に提出**
4. **都道府県が審査**（2〜3ヶ月程度）
5. **認定通知・支給開始**（認定月の翌月分から支給）
6. **毎年8月に現況届の提出**（継続支給のため必須）

### 必要書類
| 書類 | 備考 |
|------|------|
| 特別障害者手当認定請求書 | 窓口で入手 |
| 診断書 | 障害種別ごとの所定様式、医師作成 |
| 個人番号確認書類 | マイナンバーカード等 |
| 年金証書・通帳の写し（受給者のみ） | — |
| 身体・精神・療育手帳の写し（所持者） | — |

### 申請先
住所地の市区町村の障害福祉担当窓口

### 注意点
- **施設入所中・3ヶ月超の入院中は支給停止**（退所・退院後に再申請）
- **現況届（毎年8月）を忘れると支給が止まる**
- 所得制限超過の場合は「所得制限超過届」を提出し、翌年以降再確認
- 診断書様式は障害ごとに異なるため、窓口で正しい様式を入手してから医師に持参する

## 7. 介護医療院・特別養護老人ホームの入所申請

### A. 特別養護老人ホーム（特養・介護老人福祉施設）

#### 概要
要介護3以上の人が入所できる公的介護施設。低コストで24時間介護を受けられる。原則として要介護3以上が入所条件。

#### 対象者
- **要介護3以上**（例外的に要介護1・2でも特別な事情がある場合は入所可）
- 65歳以上（40〜64歳は特定疾病がある場合）

#### 手続きステップ

1. **施設を探す**
   - 市区町村の介護保険課、地域包括支援センター、介護サービス情報公表システムで検索
2. **希望施設に直接連絡・見学**
3. **入所申込書を施設に提出**
   - 複数施設への同時申込が可能（推奨）
4. **施設の入所判定会議**
   - 介護状況・生活状況・家族状況を総合評価（申込順ではない）
5. **空床発生時に施設から連絡→面接・健康診断**
6. **入所契約・入所**

#### 必要書類（施設に提出）
| 書類 | 備考 |
|------|------|
| 入所申込書 | 施設ごとの書式 |
| 介護保険被保険者証の写し | — |
| 要介護認定結果通知書の写し | — |
| 主治医意見書の写し（任意） | — |
| 健康診断書（面接後） | 施設指定の書式 |

#### 申請先
各特別養護老人ホーム（直接施設への申込）

#### 待機期間の目安
- 地方：3ヶ月未満
- 都市部（東京等）：半年〜1年以上

#### 注意点
- 入所は申込順ではなく「優先度（介護の必要性・家族状況）」で決まる
- 複数施設に同時申込が基本（1施設だけだと長期化する）
- 申込後も状況変化があれば施設に連絡して情報更新すること
- 認知症や独居等の事情を申込書の「入所理由」に具体的に記載すると優先度が上がる場合がある

---

### B. 介護医療院

#### 概要
医療的ケアが必要な要介護者向けの長期療養施設。従来の介護療養型医療施設の移行先として2018年に創設。

#### 対象者
- **要介護1以上**の認定を受けた人
- 継続的な医療処置（経管栄養・喀痰吸引等）が必要な人

#### 手続きステップ

1. **要介護認定を取得**（未取得の場合は市区町村で申請）
2. **施設を探す**（介護サービス情報公表システム等）
3. **施設に見学・相談の連絡**
4. **入所申込書を提出**
5. **医療情報の審査・面接**（医師・相談員が現在の医療ニーズを確認）
6. **入所契約・入所**

#### 必要書類
| 書類 | 備考 |
|------|------|
| 入所申込書 | 施設書式 |
| 介護保険被保険者証 | — |
| 要介護認定結果通知書 | — |
| 診療情報提供書（紹介状） | 主治医から |
| 健康保険証 | — |

#### 申請先
各介護医療院（直接施設への申込）

#### 注意点
- 入所一時金は不要（介護保険適用施設）
- 特養より医療ケアの比重が高く、胃ろう・吸引等が必要な方が対象
- 施設数が特養より少なく、地域によって選択肢が限られる

## 8. ワーキングケアラー向けその他の制度

### A. 介護休暇（育児・介護休業法）

| 項目 | 内容 |
|------|------|
| 概要 | 通院付添・手続き等の短期対応向け有給休暇 |
| 対象 | 要介護状態の家族がいる労働者 |
| 取得日数 | 対象家族が1人：年5日、2人以上：年10日 |
| 取得単位 | 1日または半日単位（時間単位は任意） |
| 申請方法 | 事業主に申出（書面）。理由を証明する書類の提出を求められる場合あり |
| 給付金 | なし（無給の場合が多い）|

### B. 短時間勤務制度・所定外労働免除・深夜業制限

| 制度 | 内容 |
|------|------|
| **短時間勤務（時短）** | 1日の所定労働時間を短縮。利用開始から連続3年間・2回以上の利用が可能（2025年改正以降） |
| **所定外労働の免除（残業免除）** | 介護が必要な家族がいる場合、残業を免除するよう申出可能。事業主は原則拒否不可 |
| **深夜業の制限** | 午後10時〜午前5時の労働制限を申請可能 |
| **転勤配慮義務** | 事業主は転勤命令の際に介護状況への配慮が義務（2025年改正） |

**申請方法：** いずれも事業主への書面申出。就業規則や会社書式を確認する。

### C. 高額医療・高額介護合算療養費制度

| 項目 | 内容 |
|------|------|
| 概要 | 医療保険と介護保険の自己負担合計が年間上限を超えた場合に払い戻し |
| 申請先 | 医療保険の窓口（健康保険組合・協会けんぽ等）が取りまとめ |
| 計算期間 | 毎年8月〜翌年7月の1年間 |
| 年間上限の目安 | 70歳未満・年収370〜770万円で67万円 |
| 申請期限 | 計算期間終了後の翌8月1日から2年以内 |

### D. 障害年金（要介護者が対象になる場合）

| 項目 | 内容 |
|------|------|
| 概要 | 病気・ケガで障害を負い生活・仕事が制限される場合に支給 |
| 種類 | 障害基礎年金（1〜2級）・障害厚生年金（1〜3級） |
| 申請先 | 年金事務所または市区町村の国民年金窓口 |
| 注意 | 初診日・保険料納付要件・障害認定日等の確認が必要 |

### E. 住宅改修費の支給（介護保険）

| 項目 | 内容 |
|------|------|
| 概要 | 手すり設置・段差解消等の小規模住宅改修に要した費用の一部を支給 |
| 支給限度額 | 20万円（うち保険給付は18万円 = 90%） |
| 対象 | 要介護・要支援認定者 |
| 申請先 | 市区町村の介護保険課（工事前に事前申請が必要） |
| 注意 | **工事前に事前申請**が必須。事後申請では支給されない場合がある |
