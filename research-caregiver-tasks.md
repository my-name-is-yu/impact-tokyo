## 1. 日常のケア業務（AIでは代替不可）

ピッチで「ここは人間の仕事。MimamoAIはここには手を出さない」と示す部分。

| タスク | 頻度 | 時間/回 | 負担の理由 |
|--------|------|---------|-----------|
| 起床・就寝介助（移乗・体位変換） | 朝晩 | 15〜30分 | 腰への物理的負荷、転倒リスク |
| 食事介助（刻み食・とろみ食の調理含む） | 3食 | 30〜60分 | 誤嚥リスク、栄養管理の複雑さ |
| 排泄介助（トイレ誘導・おむつ交換） | 1〜数時間おき | 10〜20分 | 夜間対応が睡眠を分断 |
| 入浴介助 | 週2〜3回 | 30〜60分 | 転倒・溺水リスク、体力消耗 |
| 口腔ケア | 毎食後 | 5〜10分 | 誤嚥性肺炎予防のため省略不可 |
| 服薬管理・確認 | 1日3〜5回 | 5〜15分 | 薬の種類が多い（高齢者は平均5〜7種類） |
| バイタルチェック（血圧・体温） | 1〜2回/日 | 5分 | 異変発見の責任が家族にある |
| 体位交換（寝たきりの場合） | 2時間ごと（夜間も） | 5〜10分 | 睡眠が完全に分断される |

**要介護4〜5の場合、身体介護だけで1日4〜8時間。夜間対応があれば実質24時間拘束。**

## 2. 記録・報告系タスク（MimamoAI P0で代替）

### ケア日誌（毎日 / 15〜30分）

- **記録内容:** 食事量・水分量、排泄の状態（回数・性状）、体温・血圧、服薬の有無、本人の言動・精神状態、転倒や異変の有無
- **負担の理由:**
  - 介護後に疲弊した状態で書く
  - 「何をどこまで書けばいいか」が不明確
  - 手書きノート・スマホメモ・記憶がバラバラに散在
  - 書いても誰も見ない感覚

### ケアマネへの月次報告（月1回 / 30〜60分）

- **作業内容:** 1ヶ月の状態変化をまとめ、服薬変更・医師の指示変更を書面で共有
- **負担の理由:**
  - 何を伝えるべきかわからない
  - 日々の記録が整理されていないと前日に焦る

### 医師への事前情報メモ（月1〜2回 / 20〜40分）

- **作業内容:** 通院前に「最近の変化メモ」作成（いつから何の症状か、服薬状況、転倒の有無等）
- **負担の理由:**
  - 診察は5〜10分しかない
  - 準備しないと「何か変わりましたか？」「特にないです」で終わり、最適な治療が受けられない

### ヘルパー・デイサービスへの申し送り（週2〜5回 / 10〜20分）

- **作業内容:** 体調変化・注意事項の伝達、返宅時の報告書受け取り・確認
- **負担の理由:**
  - 複数のサービス事業者に同じ情報を何度も伝える「情報の蛸足化」
  - 情報が引き継がれているか不安

## 3. 行政手続き・書類系タスク（MimamoAI P1で代替）

### 初回の介護保険申請（発生時1回 / 3〜5時間 + 30日待機）

1. 市区町村の介護保険窓口に「要介護認定申請書」を提出
2. かかりつけ医に「主治医意見書」の作成依頼
3. 認定調査員が自宅訪問（日程調整・立会い・聞き取り回答）
4. 結果通知を待つ（約30日）
5. 結果通知書・被保険者証の受け取り・保管

**負担:** 初めての人は何から始めるかわからない。認定調査で「普段より良く見せてしまい」実態より軽い認定が出るケースが多発。

### 要介護認定の更新申請（1〜3年ごと / 2〜3時間）

- 有効期限が迫ると通知 → 同様の手順を繰り返す
- **負担:** 期限を把握していないと失効し、サービスが一時停止する

### 区分変更申請（状態悪化時 / 2〜3時間 + 30日待機）

- 状態が悪化したが認定有効期間内 → 区分変更申請が必要
- 必要書類: 申請書 + 本人確認書類（再度認定調査あり）
- **負担:** 「申請できること自体を知らない」家族が多い

### 介護休業申請（職場への申請 / 1〜2時間 + 交渉）

- 必要書類: 介護休業申出書 + 要介護状態の証明書類（診断書・介護保険証等）+ 住民票
- 給付金申請: 「雇用保険介護休業給付金支給申請書」「賃金月額証明書」
- **負担:** 制度の存在は知っていても手続きが複雑。上司への説明資料を自分で作らなければならない

### その他の行政手続き

| 手続き | タイミング | 難易度 |
|--------|-----------|--------|
| 身体障害者手帳申請 | 該当状態になったとき | 高（複数機関から診断書取得） |
| 高額介護サービス費の還付申請 | 月次 | 中 |
| 特定疾病認定申請（40〜64歳対象） | 該当時 | 高（医師連携必要） |
| 成年後見制度の申立て | 判断能力低下時 | 非常に高（家庭裁判所） |
| 福祉用具の給付申請 | 必要時 | 中（ケアマネ連携必要） |

**最大の問題: 制度の情報非対称性 — 「使えるはずの支援を、存在を知らないために使えていない」**

## 4. コミュニケーション・調整系タスク

### 家族が「連絡の一元化窓口」になっている

並行して動く関係者:
- 訪問介護ヘルパー（A事業所）
- デイサービス（B事業所）
- 訪問看護（C事業所）
- かかりつけ医（D医院）
- ケアマネ（E事業所）

→ Aに伝えた情報をBにも伝え、変更があれば全員に連絡する役割を家族が担う

**1日平均30〜60分の連絡対応**（電話・FAX・手書きメモが主流）

### ケアマネジャーとのやり取り

- **月1回のモニタリング面談:** 状態報告 + サービス変更相談 + 次の計画確認（事前メモ作成が必要）
- **臨時連絡:** 入院・転倒・状態悪化のたびに電話報告（都度10〜30分）
- **サービス担当者会議:** 年1〜2回、家族も出席が求められる

### 入退院時の対応（発生時 / それぞれ1日がかり）

- **入院時:** 医師への説明、同意書署名（何枚も）、持ち物準備・名前書き、ケアマネ報告、ヘルパーキャンセル連絡
- **退院時:** 退院前カンファレンス参加、在宅復帰の受け入れ準備、在宅サービスの再手配

## 5. 情報収集・学習系タスク

### 介護保険制度の理解（初回: 数時間〜数日）

- 要介護度別の支給限度額、利用できるサービスの種類、自己負担割合（1〜3割）、地域加算の有無
- **負担:** 制度が複雑すぎて「自分が損をしているのに気づかない」ケースが多発

### 利用可能サービスの調査・比較（変更のたびに5〜10時間）

- デイサービス・訪問介護事業者の比較、空き状況確認、体験見学の手配
- **負担:** 比較情報が整備されておらず、口コミに頼らざるを得ない

### 介護技術の習得（継続的）

- 正しい移乗介助・体位変換・口腔ケアの方法を独学（YouTube・書籍）
- **負担:** 知識のない状態で試行錯誤し、介護者自身が腰痛になるケースが頻出

### 制度変更への追随（3年ごとの改正 + 随時の加算変更）

- **負担:** 「去年聞いた情報が今年は違う」が常に発生。信頼できる情報源がない

## MimamoAIが代替できる vs できない

```
代替できない（現場ケア）          代替できる（事務・情報処理）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
身体介護・入浴・食事介助      →  ケア日誌の自動生成
夜間の体位交換              →  ケアマネ向け月次報告書作成
転倒時の緊急対応            →  行政手続きの案内・書類作成補助
感情的なサポート            →  医師への事前情報メモ作成
                            →  サービス事業者への申し送り文作成
                            →  制度変更の自動アラート・解説
```

**ピッチのキーメッセージ:**
介護者は「ケアそのもの」ではなく「ケアに付随する事務作業」に潰されている。MimamoAIは事務・情報処理を丸ごとAIオーケストレーションで引き受け、介護者が「ケアだけに集中できる世界」を実現する。
